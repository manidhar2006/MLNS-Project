# HGAT_AMR Notebook Summary

This README summarizes what was implemented in `HGAT_AMR.ipynb` and the results produced in the latest executed run.

## 1) What the notebook does

The notebook builds and evaluates a **Heterogeneous Graph Attention Network (HGAT)** for TB drug resistance prediction from CRyPTIC 2024 data.

Pipeline steps:

1. Imports PyTorch Geometric, sklearn metrics/splitting, sparse linear algebra, and plotting libraries.
2. Defines 23 MTB gene regions used to map SNPs from VCFs to gene-specific node types.
3. Parses per-isolate VCF files from `./vcf(2024)` with quality and allele-frequency filters:
   - `min_qual = 20.0`
   - `min_af = 0.75`
4. Loads phenotype table `./CRyPTIC_reuse_table_20240917.csv`.
5. Selects available first-line drugs from candidates `INH`, `RIF`, `EMB`, `PZA`.
   - In this run, available columns were: `INH_BINARY_PHENOTYPE`, `RIF_BINARY_PHENOTYPE`, `EMB_BINARY_PHENOTYPE`.
6. Applies strict label mapping:
   - `R -> 1`, `S -> 0`, others (`I/NA/...`) -> `NaN`.
7. Learns SNP embeddings using PMI + truncated SVD:
   - `EMBED_DIM = 64`
   - `MIN_SNP_COUNT = 5`
8. Builds a heterogeneous graph with isolate and gene-specific SNP nodes.
9. Defines a 2-layer HGAT model (`hidden=128`, `heads=2`, dropout 0.3) and multi-label BCE training.
10. Trains with stratified split (primary stratification by RIF when available), LR scheduler, early stopping, and validation AUROC monitoring.
11. Tunes per-drug decision thresholds on validation set.
12. Evaluates on test split and generates interpretability outputs:
   - gene attention listing
   - per-isolate SNP attention
   - saved figures.

## 7) Paper-style attention interpretation (new)

The notebook now includes a paper-style interpretation block that computes per-drug attribution from learned HGAT edge attention.

### Method used

- Attribution signal: `attention × gradient` on isolate -> gene edges.
- Scope: computed on test isolates (`test_mask`) for each drug logit.
- Aggregation:
   - Per-drug gene attribution ranking.
   - Per-drug SNP attribution ranking.
   - Per-isolate, per-drug top SNP explanation.

### New notebook cells and outputs

The following outputs are written to the directory `hgat_interpretability/`:

- `{DRUG}_gene_attribution.csv`
- `{DRUG}_snp_attribution.csv`
- `mechanism_recovery_report.csv`
- `per_drug_gene_attribution.png`
- `{ISOLATE}_{DRUG}_top_snps.csv` (for the selected example isolate)

### Biological validation included

The notebook now evaluates recovery of known resistance genes for each drug using:

- `hit@k`
- `precision@k`
- `recall@k`
- hypergeometric enrichment p-value

Known-gene sets are currently defined for INH, RIF, EMB, and PZA in the validation cell and are automatically matched to available `DRUG_COLS`.

### How to run this section

1. Run all model training and test evaluation cells first.
2. Run the attribution helper cell: `Paper-style attention attribution (attention × gradient)`.
3. Run `Run per-drug attribution on test isolates`.
4. Run `Biological validation against known resistance genes`.
5. Run the plotting and single-isolate explanation cells.

This yields publication-ready per-drug attribution tables and a compact mechanism-recovery summary.
